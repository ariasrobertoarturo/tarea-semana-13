# Landing Page Interactiva

---

Sobre este proyecto...
### Este proyecto contiene implementación de interactividad avanzada: manipulación del DOM, validación de formularios y publicación con control de versiones con apoyo de IA, con temática: Gastronomía de El Salvador.
### En este proyecto Web se presenta información para un Restaurant (Pupusería "Curtido, Salsa y Sabores") que muestra menú de platos típicos, historia de los ingredientes y formulario para reserva de mesa.
